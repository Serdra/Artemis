# Artemis
A CPU-based MCTS engine with NNUE for value and policy networks. Uses UCI protocol for interacting and PUCT for search. Currently estimated at 2100 CCRL Blitz

To build on linux: Navigate to the build folder and run the following commands
```
cmake ..
cmake --build .
```
